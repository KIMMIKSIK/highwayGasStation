# highwayGasStation

- 한국도로공사의 고속도로 주유소 api를 사용하였습니다.

#### 5개의 대표 주유소를 통해 선택이 가능하고 노선명(경부선,호남선 등),방향(서울, 대구, 속초 등)을 검색하여 고속도로 각 주유소의 정보를 살펴볼 수 있습니다.
